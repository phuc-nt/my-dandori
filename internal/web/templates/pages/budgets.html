Budgets

Circuit breaker limits per scope. Changes apply to the very next tool call.

Over budget, the default downgrade mode only blocks tool calls on an expensive model (e.g. opus, fable) — the agent is told to switch model via /model and can keep working on a cheaper one; a run whose model isn't known yet is allowed through a small monthly cap before it's treated as expensive. Set budget.mode: hard in config.yaml to restore the stricter behavior: every tool call is blocked once any scope is over budget, regardless of model.
